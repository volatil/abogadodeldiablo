# Abogado del Diablo

Wen Abogao

Aplicacion Next.js para organizar tesis, objeciones, evidencia y contraargumentos.

## Stack

- Next.js con App Router
- JavaScript
- Tailwind CSS
- ESLint
- Alias de imports `@/*`

## Desarrollo

Instala dependencias y levanta el servidor local:

```bash
npm install
npm run dev
```

Luego abre `http://localhost:3000`.
